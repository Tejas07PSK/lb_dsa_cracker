# Chocolate Distribution problem

---

### Problem Links
- **_GeeksForGeeks_** :point_right: https://practice.geeksforgeeks.org/problems/chocolate-distribution-problem3825/1

### Solution
- **_solution.py_** :point_right: sort the given array, then use a sliding window of size equal to number of people, till the end of the given array, to minimize the difference between max and min chocolates given, **time-complexity O(Nlog(N))**, **space-complexity O(1)**.
